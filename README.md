## Project Description

TaskBuddy is a simple task management web application developed using HTML, CSS, and JavaScript. The application allows users to add tasks, mark tasks as completed, and delete tasks dynamically without reloading the page.

The purpose of this project is to demonstrate understanding of fundamental web development concepts including:

* HTML structure
* CSS styling
* JavaScript DOM manipulation
* Event handling
* Interactive user interface design

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Automatic task counter
* Responsive and clean design

## How to Run the Project

1. Download or clone the repository.
2. Open the file `taskbuddy.html` in any web browser.
3. The application will run immediately.
